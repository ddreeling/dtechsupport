FAQ

Do you do ladders, ceiling installs or wall cabling?

No. DTech Support currently provides desk-level work only: routers, mesh systems, devices, printers, software and configuration.

Should I buy a Wi-Fi extender before booking?

No. It is better to test the real issue first. Many homes need better placement, mesh, configuration or a router upgrade rather than a cheap extender.

Can you help with mesh Wi-Fi?

Yes. I can help set up and optimise common mesh systems such as TP-Link Deco, Eero, Google Nest Wi-Fi and similar systems.

What areas do you cover?

Cork City and County. Remote support is available anywhere.
